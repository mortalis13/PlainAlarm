## Features
- Big alarm start/stop button
- Time selection by input or sliding up/down
- Selection of a custom alarm audio file, or a folder with random audio from the local file system
- Sliders for alarm volume and snooze time

## Setup
- Remove restrictions for the battery usage (app info -> Battery -> Unrestricted and/or Battery saver -> No restrictions)
- Enable Show on lock screen and Display pop-up windows in background settings for the app
- Enable Background autostart permission (for the set alarm restoring after the device reboot)
